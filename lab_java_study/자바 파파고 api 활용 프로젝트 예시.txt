다국어 일기쓰기.

개요 - 한국어로 일기를 작성하면 다양한 언어로 번역되서 일기가 작성됌.

각각 한국어, 일본어, 중국어, 영어로 번역되어 저장.

어떤 언어로 일기를 읽고싶은지 언어를 선택하면 해당 언어의 일기가 나옴.

필요 DB

사용자DB ( 사용자넘버, 아이디, 비밀번호, 이름, 국적 )

글 DB ( 글 넘버, 글 제목, 글 내용, 외국어 DB 넘버 )

외국어 DB ( 넘버, 영어 제목, 영어 내용, 중국어 제목, 중국어 내용, 일본어 제목, 일본어 내용 )

OR

다국어 번역을 통해 외국인 직원도 사용가능한 그룹웨어 서비스
