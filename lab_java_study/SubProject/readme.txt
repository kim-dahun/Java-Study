회의실 예약 시스템

2023-03-15

1. 회의실 생성 ( Class 배열 사용, 배열 갯수 = 방 갯수 )

2. 회의실 예약

for ( 배열 생성 반복문 ) 활용해서 회의실 예약기능 생성

MainMethod - Switch( select ) => Case1

3. 예약 정보 확인 

for ( 배열 원소 체크 반복문 ) 활용해서, 배열 a1[i].name 값을 체크해서 null인지 여부에 따라 boolean 타입을 리턴하는 checkroom() 메서드 호출

호출 후 예약자가 있으면, 예약정보 출력, 없으면 예약자 없음 표시

4. 예약 취소 기능

예약자 있는 지 여부를 체크하고, 조건에 따라 있으면 예약 취소 cancelroom() 메서드 호출

없으면 문구 출력
